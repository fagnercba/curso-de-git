@ instalar o git no Windows

#Criando um repositorio#
>> Cria a pasta no PC
>> Abrir o CMD
>> Pelo cmd, abrir o diretorio/ Pasta
>> Dentro do diretorio/ Pasta, dar o comando git init <nome da pasta>

#Add e atualizar o arquivo#
>> git add <nome do projeto com a extensão>
>> git add . {este [.] add todos os arquivos para serem monitorado}
>> git status {monitora a}

OBS: dentro da pasta podemos add novos arquivos

#git reset head#
>> git reset HEAD -- <nome_do_arquivo.extensão>
>> git reset HEAD .

#git commit -m {mensagem}# historico do q foi alterado
>> git commit -m "Commit Inicial"  {mensagem para a geral}

#git checkout -b {branch}# criando a ramificação no repositório/ Pasta .ex Desenvolvimento / Teste / Produção
>> git checkout -b <nome do repositorio> // -b cria o repositorio 
>> git checkout <nome do repositorio>    // altera o diretorio/ Pasta para commitar os arquivos
>> git checkout -d <nome do repositorio> // -d deleta o repositorio 

#atualizar a branch#
>> git marge <branch> //<branch> é o repositorio que será copiado para o outro
